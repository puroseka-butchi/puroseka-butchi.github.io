{% dialogue ichika %}

{% enddialogue %}

{% dialogue saki %}
Icchan, cậu đang nghe gì vậy?
{% enddialogue %}

{% dialogue ichika %}
A, Saki……và cả Kiritani-san nữa?
{% enddialogue %}

{% dialogue ichika %}
（Kiritani-san lúc nào cũng lên sân thượng vào giờ trưa mà.
Hiếm khi thấy cậu ấy ở trong lớp thật）
{% enddialogue %}

{% dialogue haruka %}
Hoshino-san, xin lỗi vì làm phiền lúc cậu đang nghe nhạc.
{% enddialogue %}

{% dialogue ichika %}
À, không sao đâu. Dù sao cũng là bài tớ nghe suốt, nên không sao đâu.
{% enddialogue %}

{% dialogue saki %}
Vậy đúng là bài Miku-chan hát nhỉ?
{% enddialogue %}

{% dialogue ichika %}
Hả? Đúng là vậy……
{% enddialogue %}

{% dialogue saki %}
Nè nè, Haruka-chan nói là cậu ấy có hứng thú với Miku-chan đó!
{% enddialogue %}

{% dialogue haruka %}
Gần đây tớ có tự nhiên gặp Mi……không,
tớ đã xem video của Miku, rồi tự nhiên thấy tò mò.
{% enddialogue %}

{% dialogue haruka %}
Khi kể chuyện đó với Tenma-san,
cậu ấy bảo Hoshino-san rất rành về Miku.
{% enddialogue %}

{% dialogue haruka %}
Nên là, nếu được thì tớ muốn nhờ cậu chỉ cho……
{% enddialogue %}

{% dialogue ichika %}
À, ừ. Nếu thế chắc được.
Đó là video như thế nào vậy?
{% enddialogue %}

{% dialogue haruka %}
Nếu nói về thể loại thì là idol song.
Miku hát trên một sân khấu rất lớn.
{% enddialogue %}

{% dialogue haruka %}
Cứ như thể Miku thật sự đang ở đó vậy……
{% enddialogue %}

{% dialogue ichika %}
……Vậy chắc là bài được diễn trong virtual live nhỉ?
{% enddialogue %}

{% dialogue ichika %}
Nếu là bài đó thì nổi tiếng lắm…….
Để mình tìm thử nhé.
{% enddialogue %}

{% dialogue haruka %}
Cảm ơn cậu, Hoshino-san.
{% enddialogue %}

{% dialogue haruka %}
Hoshino-san có hay đến xem
virtual live không?
{% enddialogue %}

{% dialogue ichika %}
Ừ, thỉnh thoảng tớ cũng đi.
{% enddialogue %}

{% dialogue haruka %}
……Ở đó thật sự có thể nói chuyện với……
{% enddialogue %}

{% dialogue haruka %}
……Không, không có gì đâu.
Ngoài ra còn bài nào cậu muốn giới thiệu không?
{% enddialogue %}

{% dialogue ichika %}
Nếu là bài muốn giới thiệu thì nhiều lắm.
{% enddialogue %}

{% dialogue ichika %}
Những bài có hơi hướng idol song thì,
bài này……và cả bài này nữa……
{% enddialogue %}

{% dialogue haruka %}
À, bài này dễ thương thật.
{% enddialogue %}

{% dialogue ichika %}
Ngoài ra cũng có cả cover nữa.
Bài này có lẽ khá nổi tiếng.
{% enddialogue %}

{% dialogue haruka %}
À, tớ biết bản gốc bài này.
Fufu, khi Miku hát thì lại có một bầu không khí khác hẳn nhỉ.
{% enddialogue %}

{% dialogue ichika %}
Ngoài ra còn có bài như thế này nữa.
{% enddialogue %}

{% dialogue haruka %}
Hả? Miku cũng có thể hát bằng giọng như thế này sao?
{% enddialogue %}

{% dialogue haruka %}
……Miku đúng là tuyệt thật.
{% enddialogue %}

{% dialogue haruka %}
…………
{% enddialogue %}

{% dialogue ichika %}
……Kiritani-san?
{% enddialogue %}

{% dialogue haruka %}
À……xin lỗi nhé. Không có gì đâu.
{% enddialogue %}

{% dialogue haruka %}
Cảm ơn cậu. Những điều này giúp ích cho tớ nhiều lắm.
Nếu có bài nào muốn giới thiệu nữa, lần sau hãy cho tớ biết nhé.
{% enddialogue %}

{% dialogue saki %}
……Không biết bọn mình có giúp được gì cho cậu ấy không nhỉ?
{% enddialogue %}

{% dialogue ichika %}
Saki? Kiritani-san có chuyện gì sao?
{% enddialogue %}

{% dialogue saki %}
Ừm……tớ cũng không rõ, nhưng trông cậu ấy có vẻ đang băn khoăn gì đó…….
Lúc tớ lại bắt chuyện thì thành ra nói về Miku-chan.
{% enddialogue %}

{% dialogue saki %}
Nên tớ nghĩ nếu chỉ cho Haruka-chan về Miku-chan,
thì có khi chuyện khiến cậu ấy băn khoăn cũng sẽ được giải quyết.
{% enddialogue %}

{% dialogue ichika %}
Vì vậy mới dẫn đến chỗ tớ nhỉ.
{% enddialogue %}

{% dialogue saki %}
Ừm……mong là mình đã giúp được cậu ấy.
{% enddialogue %}

{% dialogue ichika %}
……Chắc chắn sẽ ổn thôi.
Nếu cậu ấy lại có vẻ băn khoăn nữa, lần sau hai đứa mình cùng bắt chuyện nhé.
{% enddialogue %}

{% dialogue saki %}
Ừm. Với lại, có một chuyện tớ hơi để ý…….
Haruka-chan vừa định nói là cậu ấy đã gặp Miku-chan đúng không?
{% enddialogue %}

{% dialogue ichika %}
Chuyện đó…… tớ cũng để ý, nhưng…
{% enddialogue %}

{% dialogue saki %}
Hay là giống tụi mình,
cậu ấy cũng đã gặp Miku-chan…… kiểu vậy?
{% enddialogue %}

{% dialogue ichika %}
Mà chắc không có đâu……
{% enddialogue %}

{% dialogue saki %}
Cũng đúng ha～
{% enddialogue %}

{% dialogue saki %}
Nè nè. Tớ cũng muốn
nghe bài Haruka-chan đã nghe～!
{% enddialogue %}

{% dialogue ichika %}
Rồi rồi. Chờ một chút nhé.
{% enddialogue %}
